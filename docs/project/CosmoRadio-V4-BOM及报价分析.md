## 二、单套 BOM（V4 USB HID + NFC）

### V3 → V4 变化对比


| 组件  | V3                        | V4                            | 变化原因                                           |
| --- | ------------------------- | ----------------------------- | ---------------------------------------------- |
| 主控  | ESP32-S3 SuperMini ¥20.68 | ESP32-S3 DevKitC N16R8 ¥32.75 | SuperMini 仅 11 GPIO，NFC SPI 需 5 pin，总需 14+ pin |
| 按钮  | 轻触按键 12×12 ¥0.15          | Kailh BOX 轴 + 6.25U 卫星轴 ~¥12  | 蝴蝶结构，≥3mm 行程，稳定不卡                              |
| NFC | 无                         | RC522 mini 模块 ¥9.94           | 新增功能，读 NTAG213 UID                             |
| PCB | 万能板 ¥0.65                 | 定制 PCB (JLCPCB) ~¥3/片         | 消灭飞线，XH2.54 即插即用                               |
| 连接器 | 焊接直连                      | XH2.54 插座 + 端子线 ~¥6           | 模块化，零焊接组装                                      |
| 螺丝  | M2 少量                     | M2.5 × 8 颗 + 热熔螺母 ~¥1         | 4 螺丝固定，结构加强                                    |
| USB | 无                         | PCB 双 USB-C 座 + VBUS 注入 ~¥9    | 板载充电+数据分流，替代 FPC+一分二（[方案](../hardware/usb.md)）          |
| 外壳  | 一体两件式 ~300g               | 模块化三件式 ~350g                  | 主体 + 顶部 + 背板                                   |


> **芯片选型结论**：
>
> - 只有 ESP32-S2 和 S3 支持 USB OTG (TinyUSB HID)，C3/C6/H2 的 USB 是 Serial/JTAG 硬件锁定，**不可用于 HID**
> - ESP32-S3 SuperMini (¥20.68) 仅 11 GPIO，V4 新增 NFC SPI (5 pin) 后总需 14+ pin，**GPIO 不够**
> - ESP32-S3 DevKitC N16R8 (¥32.75 zave) 36+ GPIO，满足所有外设 + 未来扩展
> - 备选供应商：深圳优信电子 (淘宝)，SuperMini 系列更便宜，可作为备件来源
