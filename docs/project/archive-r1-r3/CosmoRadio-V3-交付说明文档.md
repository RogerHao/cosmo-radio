## 二、项目概述

Cosmo Radio 是一款桌面交互设备（"平行宇宙收音机"），产品形态为安卓平板 + 3D 打印外壳 + ESP32 HID 控制器 + 旋钮/按钮，用户通过物理旋钮操控平板上运行的安卓应用。

第二期（2025 Q4）基于 E5 Ultra 安卓游戏掌机完成了小方盒子形态的原型验证与 BLE HID 固件开发。

第三期核心工作为：基于全新的 7 寸安卓平板形态进行结构重新设计、物料采购、批量组装与交付，并在交付过程中完成了从 BLE（蓝牙）到 USB（有线）的 HID 方案升级。

## 七、备注

1. **方案变更说明**：原合同约定交付 8 套含平板设备（BLE 蓝牙方案）。因三星 Tab A9 单台成本约 ¥1,136（美国 Amazon 价格），经协商调整为 5 套含平板设备。后因蓝牙连接稳定性不足，双方于 2026.02.03 确认切换为 USB HID 有线方案，并额外交付 7 套 USB 版本控制器/组件供甲方升级替换。

2. **平板归属**：全部 5 台三星 Galaxy Tab A9 平板随第一批设备交付甲方。第二批交付中的 1 台含平板设备使用的是第一批中回收的测试平板。

3. **后续支持**：USB HID 固件已完成验证，甲方可自行将第一批 BLE 版本设备升级为 USB 版本（更新控制器固件+使用USB接入平板）。
