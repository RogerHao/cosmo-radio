## 二、打印方案

整套外壳分为 **两个打印任务**，每个任务一盘打完：

### 打印任务一：底座

包含零件：
- 底座主体 × 1
- 底座脚垫 × 4
- 旋钮帽 × 2
- 充电线卡扣 × 1

### 打印任务二：外壳

包含零件：
- 外壳主体 × 1
- 外壳后盖 × 1
- 电源按钮 × 1
- 屏幕卡扣 × 1

## 三、打印参数

| 参数 | 设定值 |
|------|--------|
| 打印机 | Bambu Lab P2S |
| 喷嘴直径 | 0.4mm |
| 层高 | 0.2mm |
| 材料 | PLA |
| 热床类型 | 纹理 PEI 板（Textured Plate） |
| 填充密度 | 15%（默认） |
| 支撑 | 工程文件中已配置，无需额外设置 |

---

## 四、打印操作步骤

### 方式一：直接打印（推荐）

使用已切片的 `.gcode.3mf` 文件，无需任何参数调整：

1. 打开 **Bambu Studio**
2. 点击 **文件 → 打开项目**，选择 `CosmoRadio-V3-底座打印.gcode.3mf`
3. 确认打印机已连接，点击 **发送打印**
4. 底座打印完成后，同样操作打印 `CosmoRadio-V3-外壳打印.gcode.3mf`

### 方式二：修改参数后打印

如需调整打印参数（材料颜色、填充密度等）：

1. 打开 **Bambu Studio**
2. 点击 **文件 → 打开项目**，选择 `.3mf` 工程文件（非 gcode 版本）
3. 根据需要修改参数
4. 点击 **切片**，确认无误后 **发送打印**

## 五、单套用量与批量打印

| 项目 | 数量 |
|------|------|
| 每套设备打印任务数 | 2 次（底座 + 外壳） |
| 每套预估 PLA 用量 | 约 300g（底座 ~200g + 外壳 ~100g） |
| 每套预估打印时间 | 约 8-12 小时（两次打印合计） |

批量打印 N 套设备，需执行 2×N 次打印任务。

---

## 六、注意事项

1. **首次打印建议**：先各打印一套，确认尺寸和装配无误后再批量打印
2. **热床粘附**：使用纹理 PEI 板，打印前确保热床清洁
3. **取件**：等待热床冷却至室温后取件，避免变形
4. **存放**：PLA 耗材开封后注意防潮，建议使用干燥箱存放
5. **装配顺序**：先装底座（安装控制器、旋钮），再扣合外壳
