## 1. Claude Code Token 成本

口径：本项目目录下全部 9 个 Claude Code session（`~/.claude/projects/-Users-dehonghao-Projects-cosmo-radio/*.jsonl`），全部使用 `claude-opus-4-7`。

### 1.1 总量与等值成本

| token 类别 | 数量 | Opus 4.7 单价 (/MTok) | 等值成本 (USD) |
|------------|------|----------------------|----------------|
| 未缓存 input | 5,656 | $15.00 | $0.08 |
| output | 3,078,063 | $75.00 | $230.85 |
| cache write | 8,127,190 | $18.75 | $152.38 |
| cache read | 322,457,959 | $1.50 | $483.69 |
| **合计** | **333.7M** | — | **≈ $867 / ¥6,156**（@7.1）|

> **关键口径说明**：
> 1. **这是 API 等值价值，不是实际现金支出**。Claude Code 跑在 Anthropic 订阅（包月固定费）上，边际现金成本 ≈ 0。本表回答的是"如果按 API 计费，这些 token 值多少钱"。
> 2. 333.7M token 里 **322M 是 cache read**（占 97%），单价仅为 input 的 1/10。项目几乎全程命中提示缓存——若误按 input 单价统计会把数字虚高 10 倍。
> 3. 第 9 个 session（本次核算 session）仍在进行中，快照偏小，实际略高于上表。

## 2. 时间投入

| 维度 | 数值 | 口径 |
|------|------|------|
